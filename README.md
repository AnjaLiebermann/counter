# Countdown Timer

A simple countdown timer application built with Kotlin and Vaadin.

## Features

- Enter a number of minutes (1-999)
- Start the countdown
- Display shows time in MM:SS format
- Stop button to cancel the countdown
- Notification when countdown finishes

## Running the Application

```bash
./gradlew bootRun
```

Then open your browser at: http://localhost:8080

## Requirements

- Java 21
- Gradle 8.14 (included via wrapper)
